### How to run the code 
``` docker-compose up ```

### How to check the fastapi endpoint
Open in the browser
``` http://localhost:9001 ```

### How to check if haproxy is working 
``` docker-compose stop dynomite1 or dokcer-compose stop dynomite2 ```
